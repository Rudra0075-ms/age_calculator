# 🎂 Age Calculator

A simple Python project that calculates a person's exact age in **years, months, and days** based on their birthdate.

## ✨ Features

* Calculates exact age
* Accepts date in `DD/MM/YYYY` format
* Handles invalid dates and future birthdates
* Beginner-friendly code

## 🛠️ Requirements

* Python 3.x
* `datetime` module (built into Python)

No additional installation is required.

## ▶️ How to Run

```bash
python age_calculator.py
```

Enter:

* Birthdate (DD/MM/YYYY)

```text
