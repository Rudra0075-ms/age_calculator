# 🎂 Age Calculator (Python)

A simple Python program that calculates a person's exact age in **years, months, and days** based on their birth date.

## ✨ Features

- 📅 Accepts birth date in **DD/MM/YYYY** format.
- 🧮 Calculates age in:
  - Years
  - Months
  - Days
- ✅ Validates future dates.
- ⚠️ Handles invalid date formats gracefully.
